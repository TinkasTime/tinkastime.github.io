CSS lernen

HTML gibt die Struktur der Website vor und CSS ist für das designen da.

So oder so ähnlich lautet der erste Satz in diversen Büchern und Internetseiten. Deswegen sieht meine Website auch noch so simpel aus, weil sie bis jetzt nur aus einem Gerüst aus Überschrift, Paragraphen und Links besteht.

Neben HTML ist CSS der zweite Schritt, um eine Website zu bauen. Jetzt da die Struktur in HTML steht, kommt jetzt das Design hinzu. Dabei lässt sich jedes HTML-Objekt einzeln designen.

Als erstes würde ich gerne auf jeder Website auf der linken Seite eine Art Inhaltsverzeichnis basteln, sodass man von jeder Website auf alle anderen zugreifen kann.

CSS Dateien erstellen und nutzen

Genau so wie HTML-Dateien lassen sich CSS-Dateien einfach erstellen. Wichtig ist aber, dass die Datei nicht mit .html, sondern mit .css endet. Da es für kleinere Websites doch übersichtlicher sein kann, empfehlt es sich erstmal alles in einer main.css Datei zu speichern.

Um jetzt die CSS-Datei auf unsere HTML-Datei anwenden zu können, muss in der &lnhead>-Tag ein <link rel="stylesheet" href="main.css"> stehen. Hierbei muss die main.css Datei im selben Ordner wie die HTML-Datei sein.

Das Inhaltsverzeichnis

Zur Zeit der Erstellung des Inhaltsverzeichnisses gibt es nur drei Seiten: Homepage, HTML lernen und CSS lernen.
Die Homepage sollte eine Kategorie werden und die eigentliche Website eine "Über mich".
HTML und CSS lernen sollen unter der Kategorie Tutorials gelistet sein.

CSS kann unterschiedlich viele Elemente gleichzeitig bearbeiten.
Ich könnte also mit ul{ } alle unordered list auf der Website auf einmal bearbeiten.
Ich könnte aber auch nur ausgewählten Elementen eine Klasse class="speziell" geben und dann mit .speziell eben nur diese verändern.
Oder mit einer ID id="einmalig" auf ein einzelnes Element mit dieser ID anpassen.
Da das Inhaltsverzeichnis ein einzigartiges Element ist, habe ich die ul class="Inhaltsverzeichnis" definiert.

Ich möchte, dass die Liste ganz links am Rand liegt. Dafür habe ich erstmal die Stichpunkte und das padding entfernt. Weil ich die Breite nicht in Prozent oder Pixelbreite angeben wollte, habe ich sie jetzt auf die Breite des Inhalts gesetzt. Die Umrandung diente nur als visuelles Feedback, um zu schauen, wie sich width mit unterschiedlichen Parametern ändert:
.Inhaltsverzeichnis {
list-style-type: none;
padding: 0;
width: fit-content;
border: 1px solid black;
}

Weiter möchte ich die Listenelemente in zwei Arten einteilen: Einmal mit der Klasse "Inhaltsverzeichnis-Kategorie", die eine Gruppe an weiterführenden Links in eine Kategorie sammelt, leicht fett und unterstrichen sein soll, und einmal mit der Klasse "Inhaltsverzeichnis-Link", die dann die eben genannten Links sein soll. Diese sollen erstmal als einfacher Text dargestellt und mit den Kreispunkten dargestellt werden:
.Inhaltsverzeichnis-Kategorie {
font-weight: bolder;
text-decoration: underline solid black;
}
.Inhaltsverzeichnis-Link {
list-style-type: disc;
list-style-position: inside;
}
